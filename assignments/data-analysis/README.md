# 📘 Assignment: Data Analysis

## 🎯 Objective

Practice the basics of data analysis using Python by loading, exploring, and analyzing a dataset to extract meaningful insights.

## 📝 Tasks

### 🛠️	Data Loading and Exploration

#### Description
Load a provided CSV dataset and perform basic exploration to understand its structure and contents.

#### Requirements
Completed program should:

- Load a provided CSV file using Python (for example, with pandas).
- Display the first 5 rows of the dataset.
- Show summary statistics for numeric columns.


### 🛠️	Data Visualization and Insights

#### Description
Create visualizations to help understand the data and summarize key findings.

#### Requirements
Completed program should:

- Generate at least two different plot types (for example, histogram and scatter plot).
- Identify and describe at least two insights or trends from the data.
- Save generated plots as image files.
