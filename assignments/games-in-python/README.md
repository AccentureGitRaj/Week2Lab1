# 📘 Assignment: Hangman Game Challenge

## 🎯 Objective

Practice core Python fundamentals by building a classic Hangman game. You will use strings, loops, conditionals, and user input to create a playable terminal experience.

## 📝 Tasks

### 🛠️	Create the Game Setup

#### Description
Set up the core game data and initial state before gameplay begins.

#### Requirements
Completed program should:

- Define a predefined list of possible words.
- Randomly select one word from the list.
- Create variables to track guessed letters and remaining incorrect attempts.
- Display the hidden word in placeholder format (for example: `_ _ _ _`).


### 🛠️	Implement the Game Loop

#### Description
Build the interactive game loop that accepts guesses, updates game state, and ends with a clear result.

#### Requirements
Completed program should:

- Prompt the player to enter one letter per turn.
- Reveal correctly guessed letters in the word display.
- Decrease remaining attempts for incorrect guesses.
- End the game when the word is fully guessed or attempts reach zero.
- Display a clear win or loss message at the end of the game.
